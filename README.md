# Spec Taggerz

**Spec Taggerz** is a fun web-based game where players can run, jump, and enjoy gameplay. The website features a showcase of game images, a download link, and a simple comment system with user authentication via username.

---

## Features

- Showcase of game images
- Download button linking to the game
- Simple user authentication via username
- Comment section with real-time updates (requires Firebase setup)
- Styled with custom CSS for an engaging look

---

## Getting Started

### Prerequisites

- A web server or hosting platform to deploy your site
- (Optional) Firebase account for the comment system

### Usage

1. Clone or download the repository
2. Ensure `index.html` and `Site.css` are in the same directory
3. Replace Firebase configuration in the HTML with your own Firebase project details (if using Firebase)
4. Upload files to your hosting platform
5. Visit your site and enjoy!

---

## Customization

- Replace placeholder images with your own game images
- Update the download URL
- Customize CSS in `Site.css` to match your style preferences
- Implement additional features as needed
